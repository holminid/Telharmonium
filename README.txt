Seamless Background Package
---------------------------
Files:
  - background_full.webp  (optimized from original)
  - original.jpeg         (as provided)
  - tile_512.png / tile_512.webp
  - tile_1024.png / tile_1024.webp
  - index.html (demo page)
  - styles.css (CSS used by the demo)

Usage:
  Prefer WEBP in modern browsers; fall back to PNG if needed. Example CSS:

  body {
    background-image: url("tile_512.webp");
    background-repeat: repeat;
    background-position: center;
    background-size: auto;
  }

  /* Smaller or larger pattern */
  /* background-size: 200px 200px; */
